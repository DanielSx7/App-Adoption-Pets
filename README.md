# App-Adoption-Pets

# 🐾 App de Adoção de Pets

# Projeto de extensão - Disciplina: Desenvolvimento para Dispositivos Móveis
# Instituição: Uninassau - Análise e Desenvolvimento de Sistemas - 3 Período

## 📱 Sobre o Projeto

Este é um aplicativo mobile desenvolvido em **React Native**, com o objetivo de simular um sistema de adoção de pets.

O app permite que o usuário visualize diferentes animais disponíveis para adoção, filtre por categorias, veja detalhes de cada pet e realize uma solicitação de adoção através de um formulário.

---

## 🚀 Funcionalidades

* 🏠 Tela inicial com apresentação do app
* 📋 Listagem de pets com **FlatList**
* 🔍 Filtro por categoria (cães, gatos, hamsters e aves)
* 🐶 Tela de detalhes com informações completas
* 📜 Scroll para conteúdos maiores
* ❤️ Formulário de adoção
* 👤 Tela de perfil do usuário
* 🎨 Interface estilizada e moderna
* 🔽 Navegação com abas e stack

---

## 🛠️ Tecnologias Utilizadas

* React Native
* Expo (Snack)
* React Navigation
* JavaScript (ES6+)
* StyleSheet

---

## 🧠 Conceitos Aplicados

* Componentização
* Navegação entre telas (Stack e Tab Navigator)
* Gerenciamento de estado com `useState`
* Renderização de listas com `FlatList`
* Manipulação de eventos
* Estilização com `StyleSheet`

---


## ▶️ Como Executar

1. Acesse o projeto no Snack Expo
2. Execute no navegador ou escaneie o QR Code com o app Expo Go
3. Navegue pelo app normalmente
